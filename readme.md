# get-ip-addresses
Returns array of IPV4 Address for this machine.

## Install
~~~shell
npm i get-ip-addresses -S
~~~

## Use
~~~es6
import getIpAddresses from 'get-ip-addresses';
console.log(getIpAddresses());
~~~

~~~javascript
var getIpAddresses = require('get-ip-addresses').getIpAddresses;
console.log(getIpAddresses());
~~~
